# Optional Utility Functions

## Utils Module

Optional folder for utility functions used across the project.

### Common Utilities to Add

1. **Data Processing**
   - JSON parsing helpers
   - CSV handling
   - Data validation

2. **Formatting**
   - Output formatters
   - Report generators
   - Text utilities

3. **Logging**
   - Custom loggers
   - Log formatters
   - Error handlers

### Usage

```python
from src.utils import my_utility

result = my_utility.process_data(data)
```

To add utilities here, create new Python files in this directory.
