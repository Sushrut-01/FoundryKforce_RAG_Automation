# Optional Source Code Folders

## src/agents/
Optional folder for custom agent implementations. Use this to add specialized agents for:
- Query understanding
- Response generation
- Context retrieval
- Result aggregation

## src/utils/
Optional folder for utility functions. Use this to add:
- Helper functions
- Data processing utilities
- Formatting tools
- Common operations

To add files here:
```bash
# Create new agent
touch src/agents/my_agent.py

# Create new utility
touch src/utils/my_utils.py
```
